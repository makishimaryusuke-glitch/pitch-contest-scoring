# ピッチコンテスト 賞カテゴリー案

## 📊 評価項目ベースの賞

現在の6つの評価項目に基づいた賞を設けることで、各評価項目で優秀な作品を表彰できます。

### 1. **独創性賞（着眼点の独創性）**
- **対象**: 着眼点の独創性で最高得点を獲得した作品
- **意義**: 既存の枠にとらわれない、高校生らしい柔軟な発想やユニークな視点を表彰
- **別名案**: イノベーション賞、発想力賞、創造性賞

### 2. **当事者意識賞（背景のリアリティ）**
- **対象**: 背景のリアリティで最高得点を獲得した作品
- **意義**: 実体験や現場の課題感に基づいた強い「当事者意識」を表彰
- **別名案**: リアリティ賞、実践賞、課題発見賞

### 3. **科学的探究賞（仮説検証の適切性）**
- **対象**: 仮説検証の適切性で最高得点を獲得した作品
- **意義**: SPLYZAMotion等のデータを活用した客観的かつ科学的な検証を表彰
- **別名案**: データ活用賞、検証力賞、科学賞

### 4. **分析力賞（分析の深さ）**
- **対象**: 分析の深さで最高得点を獲得した作品
- **意義**: 「なぜそうなったのか」を深く考察し、論理的に結論を導き出した作品を表彰
- **別名案**: 論理力賞、考察賞、深掘り賞

### 5. **実践貢献賞（現場への還元）**
- **対象**: 現場への還元で最高得点を獲得した作品
- **意義**: チーム強化や競技力の向上に具体的に役立つ探究結果を表彰
- **別名案**: 実用性賞、還元賞、チーム貢献賞

### 6. **波及効果賞（波及効果）**
- **対象**: 波及効果で最高得点を獲得した作品
- **意義**: 他のチームや競技、スポーツ界全体に新しい知見や価値を提供した作品を表彰
- **別名案**: 社会貢献賞、影響力賞、拡散賞

## 🎯 総合的な賞（既存の賞以外）

### 7. **総合優勝賞**
- **対象**: 総合スコアが最高の作品
- **意義**: すべての評価項目でバランス良く優秀な作品を表彰
- **備考**: 最優秀賞と重複する場合は、最優秀賞を総合優勝賞として設定

### 8. **バランス賞**
- **対象**: 各評価項目のスコアのばらつきが最も小さい作品（標準偏差が最小）
- **意義**: すべての評価項目で均等に優秀な作品を表彰
- **別名案**: 総合力賞、オールラウンド賞

### 9. **成長賞**
- **対象**: 提出時点での完成度は低くても、探究プロセスや学びの過程が評価できる作品
- **意義**: 結果だけでなく、プロセスや成長を重視
- **備考**: 評価項目に「成長度」を追加するか、審査員の判断で選定

## 🏆 特別賞

### 10. **審査員特別賞**
- **対象**: 審査員が特に評価したい作品
- **意義**: 数値化できない価値や、審査員の専門的な視点から評価
- **備考**: 審査員の裁量で選定

### 11. **データ活用特別賞**
- **対象**: SPLYZAMotionのデータを特に効果的に活用した作品
- **意義**: データ分析の技術や工夫を表彰
- **別名案**: SPLYZAMotion賞、データ分析賞

### 12. **プレゼンテーション賞**
- **対象**: 提出資料の構成や説明が特に優れている作品
- **意義**: 伝える力や表現力を表彰
- **備考**: 当日審査機能実装後に追加検討

### 13. **チームワーク賞**
- **対象**: チーム内での協力や役割分担が評価できる作品
- **意義**: チーム活動の質を表彰
- **備考**: チーム構成情報が必要

### 14. **継続性賞**
- **対象**: 過去の取り組みを継続・発展させた作品
- **意義**: 継続的な探究活動を表彰
- **備考**: 過去のデータが必要

## 📈 統計的な賞

### 15. **飛躍賞**
- **対象**: 前回のコンテストと比較して、最もスコアが向上した作品
- **意義**: 継続的な改善と成長を表彰
- **備考**: 過去のデータが必要

### 16. **安定性賞**
- **対象**: 複数回の採点で最も一貫したスコアを獲得した作品
- **意義**: 評価の安定性を表彰
- **備考**: 複数回採点が必要

## 🎨 テーマ別の賞

### 17. **技術革新賞**
- **対象**: 技術的な側面で特に優れている作品
- **意義**: 技術的な工夫やイノベーションを表彰

### 18. **社会課題解決賞**
- **対象**: 社会的な課題に取り組んだ作品
- **意義**: 社会への貢献を表彰

### 19. **地域貢献賞**
- **対象**: 地域の課題や特色を活かした作品
- **意義**: 地域への貢献を表彰

## 💡 推奨される賞の組み合わせ

### 基本セット（推奨）
1. **最優秀賞** - 総合スコア1位
2. **優秀賞** - 総合スコア2-3位
3. **敢闘賞** - 総合スコア4-5位、または特定の評価項目で優秀
4. **奨励賞** - 総合スコア6位以下、または努力が認められる作品

### 評価項目ベースの賞（推奨）
5. **独創性賞** - 着眼点の独創性で最高得点
6. **科学的探究賞** - 仮説検証の適切性で最高得点
7. **実践貢献賞** - 現場への還元で最高得点

### 特別賞（任意）
8. **審査員特別賞** - 審査員の裁量
9. **データ活用特別賞** - SPLYZAMotionの活用が特に優れている

## 🎯 実装方法

### 自動判定可能な賞
以下の賞は、採点結果から自動的に判定できます：

- ✅ 最優秀賞（総合スコア1位）
- ✅ 優秀賞（総合スコア2-3位）
- ✅ 敢闘賞（総合スコア4-5位）
- ✅ 奨励賞（総合スコア6位以下）
- ✅ 各評価項目ベースの賞（各項目の最高得点）
- ✅ バランス賞（標準偏差が最小）

### 手動判定が必要な賞
以下の賞は、審査員の判断が必要です：

- ⚠️ 審査員特別賞
- ⚠️ 成長賞
- ⚠️ チームワーク賞
- ⚠️ 継続性賞

## 📋 賞の選定基準例

### 最優秀賞
- 総合スコアが最高
- すべての評価項目で一定以上のスコア（例：各項目7点以上）

### 優秀賞
- 総合スコアが2-3位
- または、特定の評価項目で特に優秀（例：9点以上）

### 敢闘賞
- 総合スコアが4-5位
- または、特定の評価項目で優秀（例：8点以上）
- または、努力や工夫が認められる

### 奨励賞
- 総合スコアが6位以下
- または、特定の評価項目で一定以上のスコア（例：7点以上）
- または、努力が認められる

## 💻 システムへの実装案

### 賞の自動判定機能
採点結果に基づいて、以下の賞を自動判定できます：

```python
def determine_awards(results):
    """
    採点結果に基づいて賞を自動判定
    
    Args:
        results: 採点結果のリスト
    
    Returns:
        awards: 各作品に付与された賞のリスト
    """
    # 総合スコアでソート
    sorted_results = sorted(results, key=lambda x: x['total_score'], reverse=True)
    
    awards = {}
    
    # 最優秀賞（1位）
    awards[sorted_results[0]['id']] = ['最優秀賞']
    
    # 優秀賞（2-3位）
    for i in [1, 2]:
        if i < len(sorted_results):
            awards[sorted_results[i]['id']] = ['優秀賞']
    
    # 敢闘賞（4-5位）
    for i in [3, 4]:
        if i < len(sorted_results):
            awards[sorted_results[i]['id']] = ['敢闘賞']
    
    # 各評価項目ベースの賞
    for criterion_id in range(1, 7):
        criterion_name = get_criterion_name(criterion_id)
        best_score = max(r[f'criterion_{criterion_id}_score'] for r in results)
        best_result = next(r for r in results if r[f'criterion_{criterion_id}_score'] == best_score)
        
        award_name = f'{criterion_name}賞'
        if best_result['id'] not in awards:
            awards[best_result['id']] = []
        awards[best_result['id']].append(award_name)
    
    return awards
```
