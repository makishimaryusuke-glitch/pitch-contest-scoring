# クイックスタートガイド

## 5分で始める

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. アプリの起動

**方法1: 起動スクリプトを使用（推奨）**
```bash
./start.sh
```

**方法2: 手動で仮想環境を有効化**
```bash
source venv/bin/activate
streamlit run app.py
```

ブラウザで http://localhost:8501 が開きます。

### 3. APIキーの設定（初回のみ）

アプリ起動後、「⚙️ API設定」ページで：
- OpenAI APIキーまたはGoogle Gemini APIキーを入力
- 「APIキーを設定」ボタンをクリック

**注意：** APIキーはブラウザを閉じるまで有効です。次回起動時は再度設定が必要です。

## 使い方

### 1. 参加校の登録

「📤 提出資料のアップロード」ページで、新規参加校を登録します。

### 2. 提出資料のアップロード

- 参加校を選択（または新規作成）
- テーマタイトルを入力
- PDF、PowerPoint、テキストファイルをアップロード

### 3. AI採点の実行

「🤖 AI採点の実行」ページで：
- 提出資料を選択
- 「AI採点を実行」ボタンをクリック
- 採点が完了するまで待機（1-2分程度）

### 4. 採点結果の確認

「📊 採点結果」ページで：
- レーダーチャートで評価項目別スコアを確認
- 詳細な評価理由を確認
- CSV形式でエクスポート可能

## トラブルシューティング

### データファイルエラー

データファイルが破損した場合は、`data/`フォルダ内のJSONファイルを削除すると、次回起動時に自動的に再作成されます。

### AI APIエラー

- 「⚙️ API設定」ページでAPIキーが正しく設定されているか確認
- APIキーの有効期限を確認
- APIキーの使用量制限に達していないか確認

### ファイルアップロードエラー

- `uploads/`フォルダの書き込み権限を確認
- ファイルサイズが10MB以下であることを確認
