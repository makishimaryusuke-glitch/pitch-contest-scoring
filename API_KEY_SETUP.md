# APIキー設定ガイド

## Google Gemini APIキーの取得方法

### 1. Google AI Studioにアクセス

1. ブラウザで以下のURLにアクセス：
   - https://makersuite.google.com/app/apikey
   - または https://aistudio.google.com/app/apikey

2. Googleアカウントでログイン

### 2. APIキーの作成

1. 「Create API Key」または「APIキーを作成」ボタンをクリック
2. プロジェクトを選択（新規プロジェクトを作成することも可能）
3. APIキーが生成されます
4. **重要：** APIキーをコピーして安全に保管してください

### 3. APIキーの設定（アプリ内）

1. アプリを起動
2. サイドバーの「⚙️ API設定」ページを選択
3. 「AIプロバイダーを選択」で「gemini」を選択
4. 「Google Gemini APIキー」欄にAPIキーを貼り付け
5. 「APIキーを設定」ボタンをクリック
6. ✅ が表示されれば設定完了

## OpenAI APIキーの取得方法（参考）

### 1. OpenAI Platformにアクセス

1. ブラウザで以下のURLにアクセス：
   - https://platform.openai.com/api-keys

2. OpenAIアカウントでログイン（アカウントがない場合は作成）

### 2. APIキーの作成

1. 「Create new secret key」ボタンをクリック
2. キー名を入力（任意）
3. APIキーが生成されます
4. **重要：** APIキーは一度しか表示されないので、必ずコピーして保管してください

### 3. APIキーの設定（アプリ内）

1. アプリを起動
2. サイドバーの「⚙️ API設定」ページを選択
3. 「AIプロバイダーを選択」で「openai」を選択
4. 「OpenAI APIキー」欄にAPIキーを貼り付け
5. 「APIキーを設定」ボタンをクリック
6. ✅ が表示されれば設定完了

## 注意事項

- **APIキーの管理**
  - APIキーは他人に共有しないでください
  - APIキーが漏洩した場合は、すぐに無効化してください
  - 使用量に応じて課金されるため、使用量を監視してください

- **APIキーの有効期限**
  - アプリ内で設定したAPIキーは、ブラウザを閉じるまで有効です
  - 次回起動時は再度設定が必要です

- **コスト**
  - Google Gemini: 無料枠あり（月間60リクエスト/分まで）
  - OpenAI GPT-4: 従量課金（約$0.03/1Kトークン）

## トラブルシューティング

### APIキーが正しく設定されない

- APIキーが正しくコピーされているか確認
- 前後の空白文字が含まれていないか確認
- APIキーの有効期限を確認

### API呼び出しエラーが発生する

- APIキーが正しく設定されているか確認
- APIキーの使用量制限に達していないか確認
- インターネット接続を確認
